word_cloud_maker
================

This is a word cloud generating tool that will enable people to enter some words and see it up in the cloud! A word cloud that is.

# Require and install these gems:
# require/install 'unirest'
# require/install 'JSON'
# require/install 'Launchy'

This will connect to http://mashape.com to make the word cloud and unirest is a nice gem to do so.  you can view 
# more about the unirest gem at :Mashape/unirest-ruby
